## API Token
1. go to dashboard -> Settings ⚙️ -> API -> Store-level API accounts
2. Create API Account: 
 - Token type: V2/V3 API token
 - Name: any name
 - Set permissions to at least Read-only:
  - Orders
  - Products
  - Customers
3. Copy any save API url, then click 'create' and save tokens
